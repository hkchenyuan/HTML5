1、autocomplete 属性规定 form 或 input 域应该拥有自动完成功能。
2、autofocus 属性规定在页面加载时，域自动地获得焦点。
3、form 属性规定输入域所属的一个或多个表单。
4、form override attributes允许您重写 form 元素的某些属性设定：formaction、formenctype、formmethod、formnovalidate、formtarget。
5、list 属性规定输入域的 datalist。datalist 是输入域的选项列表。
6、multiple 属性规定输入域中可选择多个值。适用于以下类型：email 和 file
7、novalidate 属性规定在提交表单时不应该验证 form 或 input 域
8、pattern 属性规定用于验证 input 域的模式（pattern）。
9、required 属性规定必须在提交之前填写输入域（不能为空）。
